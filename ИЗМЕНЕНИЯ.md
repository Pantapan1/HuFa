# Что изменилось в боте «Хуфа»

## 🔧 Технические исправления

1. **Токены больше не в коде.** BOT_TOKEN, GROQ_KEY, ADMIN_ID теперь читаются из `.env`
   рядом с `main.py`. Если файла нет или в нём чего-то не хватает — бот не запустится
   и подскажет, что заполнить. Это устраняет риск утечки токена, если код куда-то попадёт
   (например, в публичный репозиторий).
2. **Логирование вместо print.** Все системные сообщения и ошибки пишутся в `bot.log`
   рядом с ботом и одновременно выводятся в консоль — удобно искать проблемы задним числом.
3. **WAL-режим SQLite + индексы.** База данных меньше блокируется при одновременных
   запросах (например, рассылка + чей-то вопрос к вики одновременно), а частые запросы
   (по категориям вики, по историям, по балансу игроков) стали быстрее.
4. **Авто-перезапуск опроса Telegram.** Если сеть моргнёт или Telegram API отдаст ошибку,
   бот больше не «умирает» — ждёт 5 секунд и продолжает работу.
5. **Ретраи для вызовов ИИ** — добавлена функция `groq_complete()` с повторными попытками
   при сбоях Groq API (использована в новых функциях, старые генераторы не трогал, чтобы
   не рисковать текущим поведением).
6. **`requirements.txt`** — теперь понятно, что ставить: `pip install -r requirements.txt`.
7. Данные из старой базы (`hupha_bot.db`) сохранены — просто добавились новые таблицы.

## 🛒 Новое для игроков

- **Магазин** («🛒 Магазин») — админ добавляет товары, игроки покупают за хуфу/рубли.
- **Инвентарь** («🎒 Инвентарь») — что игрок купил.
- **Ежедневный бонус** («🎁 Ежедневный бонус») — раз в 24 часа случайные 20–60 рублей.
- **Топ игроков** («🏆 Топ игроков») — рейтинг по хуфе и рублям.
- **Закладки по вики** — `/bookmark ключ` сохраняет статью, кнопка «🔖 Закладки» показывает список.
- **Быстрый поиск** — `/search запрос` ищет по всем ключевым словам вики (не только точное совпадение).
- Профиль и приветствие `/start` теперь автоматически заводят игрока в базе (не нужно ждать `/create`, чтобы начать копить валюту).

## 🛠 Новое для админа

Новая **«🛠 Админ-панель»** в главном меню:

- **💾 Бэкап БД** — присылает файл базы данных прямо в чат.
- **📈 Общая статистика** — игроки, записи вики, истории, связи, РП-сессии, бан-лист, товары.
- **🚫 Бан-лист** + команды `/ban ID [причина]` и `/unban ID` — заблокированные не долбят бота.
- **💰 Выдать валюту** — `/give ID количество хуфа|рубли`.
- **🏷 Добавить товар** — `/additem Название | Цена | хуфа|рубли | Описание | Эмодзи`.
- **📤 Экспорт вики** — выгружает весь лор в `.txt`-файл одним нажатием.
- **`/recap`** — ИИ кратко пересказывает последние события активной РП-сессии
  (удобно, если мастер отвлёкся или нужно напомнить игрокам, что происходит).

## Как запустить

```bash
pip install -r requirements.txt
python3 main.py
```

Убедитесь, что `.env` рядом с `main.py` заполнен:

```
BOT_TOKEN=...
GROQ_KEY=...
ADMIN_ID=...
```

## Что не трогал (и почему)

Основную логику вики-поиска, РП-сессий, генераторов NPC/квестов/загадок и т.д. я не переписывал —
она рабочая и рискованно менять 2000+ строк одним заходом. Если хочешь, в следующий раз могу:
- разбить `main.py` на модули (handlers/, db/, ai/) — сейчас всё в одном файле;
- добавить постоянное хранение состояний (сейчас `user_states` живёт в памяти и слетает при рестарте);
- прикрутить оплату между игроками (`/pay`) или крафт предметов.
